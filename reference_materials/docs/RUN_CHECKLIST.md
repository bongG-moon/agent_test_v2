# 실행 체크리스트

이 문서는 앱을 실행한 뒤 무엇을 확인해야 하는지 단계별로 정리한 문서입니다.  
처음 실행하는 사람도 이 순서대로 따라가면 기본 동작과 도메인 등록 기능을 빠르게 점검할 수 있습니다.

## 1. 실행

```bash
cd C:\Users\qkekt\Desktop\AGENT_COMPACT_LANGGRAPH
streamlit run app.py
```

## 2. 첫 화면 확인

정상이라면 왼쪽 메뉴에서 아래 두 항목이 보여야 합니다.

- `채팅 분석`
- `도메인 관리`

## 3. 채팅 분석 화면 확인

`채팅 분석`을 선택합니다.

정상이라면 아래 요소가 보여야 합니다.

- 페이지 제목
- `ENG'R 모드` 토글
- 채팅 입력창

## 4. 기본 조회 테스트

질문:

```text
오늘 DA공정 생산량 알려줘
```

기대 결과:

- `production` 데이터가 조회됩니다.
- `DA`가 `D/A1` ~ `D/A6` 범위로 해석됩니다.
- 응답과 결과 표가 보입니다.

## 5. 복수 데이터셋 조회 테스트

질문:

```text
오늘 생산과 목표를 같이 보여줘
```

기대 결과:

- `production`
- `target`

두 데이터셋이 함께 사용됩니다.

## 6. 파생 지표 테스트

질문:

```text
어제 DA공정에서 DDR5제품 생산 달성률 알려줘
```

기대 결과:

- `production`, `target`이 함께 조회됩니다.
- 생산만 설명하고 끝나지 않고 실제 달성률 계산이 포함됩니다.

## 7. 첫 질문에서 후처리까지 되는지 테스트

질문:

```text
오늘 DA공정에서 MODE별 생산량 알려줘
```

기대 결과:

- 단순 원본 데이터만 나오지 않습니다.
- MODE별 집계된 결과 표가 바로 나옵니다.

## 8. 후속 분석 테스트

순서:

1. `오늘 DA공정 생산량 알려줘`
2. `MODE별로 묶어줘`
3. `상위 5개만 보여줘`

기대 결과:

- 2번과 3번은 새 데이터 조회가 아니라 현재 결과를 기준으로 후처리됩니다.

## 9. ENG'R 모드 테스트

`ENG'R 모드`를 켜고 아래 질문을 합니다.

```text
오늘 DA공정에서 MODE별 생산량 알려줘
```

기대 결과:

- 분석 처리 요약이 보입니다.
- 어떤 방식으로 pandas 코드가 생성되었는지 보입니다.
- 생성된 pandas 코드가 보입니다.

## 10. 도메인 관리 화면 확인

왼쪽 메뉴에서 `도메인 관리`를 선택합니다.

정상이라면 아래가 보여야 합니다.

- 페이지 제목 `Domain Knowledge Manager`
- `등록`, `목록` 탭
- 예시 안내 영역

## 11. 데이터셋 표현 등록 테스트

등록 문장:

```text
양품률이라는 표현은 수율 데이터셋을 뜻해.
```

절차:

1. `미리보기`
2. 구조화 결과 확인
3. `등록`

등록 후 질문:

```text
오늘 DA공정 양품률 알려줘
```

기대 결과:

- `yield` 데이터셋이 선택됩니다.

## 12. 필터 별칭 등록 테스트

등록 문장:

```text
후공정A는 D/A1, D/A2를 뜻해.
```

등록 후 질문:

```text
오늘 후공정A 생산량 알려줘
```

기대 결과:

- `process_name`이 `D/A1`, `D/A2`로 확장됩니다.

## 13. 계산 규칙 등록 테스트

등록 문장:

```text
홀드 부하지수는 hold 테이블의 hold_qty 와 production 테이블의 production 을 같이 가져와서
hold_qty / production 으로 계산해줘.
```

등록 후 질문:

```text
오늘 DA공정 홀드 부하지수 알려줘
```

기대 결과:

- `hold`, `production`이 함께 조회됩니다.
- 최종 결과에 홀드 부하지수 계산이 반영됩니다.

## 14. 판정 규칙 등록 테스트

등록 문장:

```text
장비 가동 이상여부는 equipment 테이블의 가동률이 85 미만이면 이상,
아니면 정상으로 판단해줘.
```

등록 후 질문:

```text
오늘 DA공정 장비 가동 이상여부 알려줘
```

기대 결과:

- `equipment`가 조회됩니다.
- 가동률 기준 판정 결과가 보입니다.

## 15. 충돌 테스트

등록 문장:

```text
수율이라는 표현은 생산 데이터셋을 뜻해.
```

기대 결과:

- `미리보기`에서 경고 또는 오류가 보여야 합니다.
- 그대로 저장하면 안 되는 상황인지 확인할 수 있어야 합니다.

## 16. 삭제 기능 테스트

절차:

1. `도메인 관리 > 목록`으로 이동합니다.
2. 방금 만든 테스트 엔트리를 찾습니다.
3. `삭제` 버튼을 누릅니다.

기대 결과:

- 성공 메시지가 보입니다.
- 목록에서 해당 엔트리가 사라집니다.

## 17. 종료

Streamlit을 종료하려면 실행한 터미널에서 `Ctrl + C`를 누르면 됩니다.  
Windows 환경에서는 네이티브 런타임 메시지가 같이 보일 수 있지만, `Ctrl + C`로 직접 종료한 상황이라면 보통 큰 문제는 아닙니다.
